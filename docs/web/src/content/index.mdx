# Introduction

**Gormite** is a ORM with [AST](https://pkg.go.dev/go/ast) based migrations
generator and query builder.

It supports many databases and nested structs mapping
using [Go Struct Tags](https://pkg.go.dev/github.com/fatih/structtag),
check [Quick start](/docs/get-started#quick-start-with-goose),
for goose usage example.
